# 智能农业监测系统 - 大创项目网站

这是一个基于HTML、CSS和JavaScript从零开始构建的大创项目展示网站。网站展示了"智能农业监测系统"项目的详细信息、团队成员、成果展示和联系方式。

## 项目特色

- 🎨 **现代化设计**：采用响应式设计，支持各种设备屏幕尺寸
- ⚡ **流畅交互**：包含平滑滚动、图片轮播、表单验证等交互功能
- 📱 **移动友好**：完全适配手机、平板和桌面设备
- 🚀 **性能优化**：代码结构清晰，加载速度快

## 技术栈

- **HTML5**：语义化标签，结构清晰
- **CSS3**：现代布局（Flexbox/Grid），动画效果
- **JavaScript**：DOM操作，事件处理，表单验证

## 功能特性

### 已实现功能
1. **响应式导航栏** - 固定顶部导航，支持平滑滚动
2. **项目介绍区** - 详细的项目背景、目标和技术路线
3. **团队成员展示** - 美观的卡片式布局，悬停效果
4. **成果轮播展示** - 自动轮播，支持手动切换
5. **联系表单** - 实时验证，友好的错误提示
6. **滚动高亮** - 导航栏自动高亮当前区域

### 交互功能
- 平滑滚动到页面指定部分
- 图片轮播自动播放和手动控制
- 表单输入的实时验证
- 成功/错误消息提示
- 移动设备触摸友好的界面

## 快速开始

### 本地运行
1. 下载所有文件到本地文件夹
2. 用浏览器打开 `index.html` 文件
3. 网站将在本地运行，可以查看所有功能

### 自定义修改

#### 修改项目信息
编辑 `index.html` 文件中的相关内容：
- 修改 `<title>` 标签中的网站标题
- 更新项目介绍、团队成员等信息
- 调整导航菜单项

#### 修改样式
编辑 `styles.css` 文件：
- 修改颜色主题（搜索 `#2c3e50` 和 `#3498db`）
- 调整字体、间距等样式属性
- 自定义动画效果

#### 添加功能
编辑 `script.js` 文件：
- 添加新的交互功能
- 修改表单验证规则
- 扩展轮播功能

## 部署到互联网

### 使用GitHub Pages（推荐）
1. 在GitHub创建新仓库
2. 上传所有文件到仓库
3. 在仓库设置中启用GitHub Pages
4. 访问 `https://你的用户名.github.io/仓库名`

### 其他托管平台
- **Netlify**：拖拽部署，支持自定义域名
- **Vercel**：专注于前端部署，自动CI/CD
- **Firebase Hosting**：Google提供的免费托管服务

## 开发建议

### 代码规范
- 使用语义化的HTML标签
- CSS类名使用BEM命名规范
- JavaScript代码模块化组织
- 添加必要的注释说明

### 性能优化
- 压缩图片资源
- 合并CSS/JS文件
- 使用CDN加载第三方库
- 启用Gzip压缩

### 可访问性
- 为图片添加alt属性
- 确保足够的颜色对比度
- 支持键盘导航
- 添加ARIA标签
