# 图片资源文件夹

此文件夹用于存放网站所需的图片资源。

## 建议的图片文件

### 必需图片
- `tech-roadmap.png` - 项目技术路线图
- `project-overview.jpg` - 项目概览图
- `system-architecture.png` - 系统架构图

### 团队成员头像（可选）
- `member-zhangsan.jpg` - 张三的头像
- `member-lisi.jpg` - 李四的头像
- `member-wangwu.jpg` - 王五的头像
- `member-zhaoliu.jpg` - 赵六的头像

### 项目展示图片
- `hardware-prototype.jpg` - 硬件原型照片
- `software-interface.png` - 软件界面截图
- `field-testing.jpg` - 实地测试照片

## 图片规格建议

- **格式**：PNG（图标、图表）、JPG（照片）
- **尺寸**：宽度不超过1200px
- **文件大小**：单张图片不超过500KB
- **分辨率**：72-150 DPI（网页使用）

## 替代方案

如果暂时没有实际图片，可以使用：
1. 占位符图片（网站已使用emoji作为头像占位符）
2. 在线工具生成的示意图
3. 相关领域的免费素材图片

## 优化建议

1. 使用图片压缩工具减小文件大小
2. 为重要图片提供WebP格式以提升加载速度
3. 为所有图片添加有意义的alt属性描述
4. 使用响应式图片（srcset属性）适配不同屏幕
